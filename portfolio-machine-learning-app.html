Application machine learning

J'ai réalisé ce projet web en groupe au cours de ma formation de type POEC de Data Analyst, pour proposer un service basique de machine learning.

L'objectif était de permettre à un utilisateur de sélectionner un fichier CSV (avec des données déjà préparées) et de tester avec quelques modèles enregistrés.

A l'aide de GridSearchCV, l'utilisateur pourra jouer avec ces modèles, avec ou sans modifier les hyperparamètres, et ainsi voir les métriques et les visualisations associées.

Technologies utilisées : Python, Matplotlib, NumPy, Seaborn, Scikit-Learn, Streamlit
