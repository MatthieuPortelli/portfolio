Site web e-commerce

J'ai réalisé ce projet web en Python, selon de nombreux tutoriels, pour simuler un site e-commerce standard commandé par un client.

Les objectifs étaient de permettre aux utilisateurs de s'inscrire (confirmation par email), de se connecter, de parcourir les catégories, les produits et de commander ces derniers.

Lors de l'achat, ils ont également la possibilité de modifier les quantités des articles, d'indiquer une adresse de livraison, puis de consulter les commandes passées dans leur espace personnel.

L'administrateur du site a la possibilité de modifier facilement les produits en ligne, et notamment leur stock.

Technologies utilisées : HTML, CSS, Python, Django, Bootstrap, SQLite, PythonAnywhere
