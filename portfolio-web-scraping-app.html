Application web scraping

J'ai réalisé ce projet web après ma formation de type POEC de Data Analyst, pour proposer un service de web scraping.

L'objectif était de permettre à un utilisateur de rentrer une url et de voir les éléments présents sur cette page.

En raison de la politique de l'hébergeur de ce projet (PythonAnywhere), l'utilisateur est soumis à certaines restrictions en ce qui concerne les sites à scraper.

Technologies utilisées : HTML, CSS, JavaScript, Python, Flask, BeautifulSoup, Matplotlib, SQLite, PythonAnywhere
